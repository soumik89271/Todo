📝 To-Do List App
A simple Dark Mode To-Do List web application built using HTML, CSS, and JavaScript. It allows users to manage daily tasks efficiently with features like task creation, editing, deletion, filtering, and local storage persistence.

🚀 Live Demo
👉 https://soumik89271.github.io/Todo/

✨ Features
➕ Add new tasks
✏️ Edit existing tasks
❌ Delete tasks
✅ Mark tasks as completed
🔍 Filter tasks (All / Completed / Pending)
📊 Task counter showing remaining tasks
💾 Data stored using LocalStorage
⌨️ Add tasks using the Enter key
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
LocalStorage API
